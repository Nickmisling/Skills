# Example Skill

This is a starter skill that demonstrates the structure of a plugin-packaged
Agent Skill in this marketplace. Replace the body below with your own
instructions.

When invoked:

1. Look at the files the user has selected or the recent uncommitted changes.
2. Summarize what changed in a few concise bullet points.
3. Call out anything that looks risky, unfinished, or worth a second look.

Keep the output short and actionable.
